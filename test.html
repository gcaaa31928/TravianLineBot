Hints to help you get a great screenshot:

  • Don't use password protected links or URLs that require login
  • Don't use local file links (file:// or c:/...) won't work
  • Prefer the http:// version (if available) for web site using self-signed certificates.
  • Local URL (http://127.0.0.1 for example) won't work
  • Use JPEG format for normal-quality screenshots and PNG format for high-quality screenshots

JPEG typically provides about the same screenshot quality as PNG, to the naked eye, but JPEG usually has a 30%-60% smaller file size compared to PNG at the same resolution.
